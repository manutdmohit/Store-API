Backend for store api.  
- Create a .env file in the root and add
```
MONGO_URI=YourConnectionString
```


- To run the project
```
  - npm install
  - npm start
```
- For Seeding data to the DataBase
 ```
 - Check seeder.js file
 To import data:
 node seeder -i 
 
 To destroy data:
 npm seeder -d
 
 ```
